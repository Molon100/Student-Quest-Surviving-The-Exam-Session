# Student Quest:Surviving The Exam Session

The following repository contains the program code file and other files required

to run Student Quest - a game project for SU-FMI Intoduction to programming

### How to run

Compile and run Student Quest.cpp

### Stats

* Energy

* Money
* Knowledge
* Psyche

### Rules

Every day you will be able to choose to do an action. Your actions will change the stats.

It's possible for your actions to not be fully effective, if you're too tired.

If you run out of either Money or Psyche, the game will end.

It's also possible to get random events after each day and action.

The main objective of this game is to pass all 5 of your exams.

### Features

* You can choose the difficulty: easy, normal and hard
* Autosave and manual save function
* Loading function
